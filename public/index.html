Welcome to the School Management System API. This is a RESTful API built with Express.js, MongoDB, and Redis.

Quick Start

Base URL: http://localhost:3000/api

Available Endpoints

POST /api/auth/register Public

Register a new user

POST /api/auth/login Public

Login and get JWT token

GET /api/schools Protected

List all schools

GET /api/classrooms Protected

List all classrooms

GET /api/students Protected

List all students
